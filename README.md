XmlViewer
=========

Usage:
------

```sh
java -jar XmlViewer-x.x.jar
```


Requirements:
-------------

- java version "9.0.1" or later.


Feature :
---------

- [ ] : View DOM tree in tree view.
- [ ] : Customizable tree view
